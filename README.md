# AutoВыкуп — современный лендинг

## Что скачать
Скачайте всю папку `auto-vykup-modern` или архив `.zip`.

## Что установить
1. Установите Node.js: https://nodejs.org
2. Установите VS Code: https://code.visualstudio.com

## Как запустить сайт локально
Откройте папку `auto-vykup-modern` в VS Code.

В терминале выполните:

```bash
npm install
npm run dev
```

Откройте в браузере:

```txt
http://localhost:3000
```

## Куда вставлять текст и менять данные
Главный файл сайта:

```txt
app/page.tsx
```

Там можно поменять:
- название компании;
- телефон;
- ссылку WhatsApp;
- условия;
- список автомобилей;
- тексты блоков;
- FAQ.

## Где менять стили
Основные стили находятся прямо в классах Tailwind внутри:

```txt
app/page.tsx
```

Глобальные стили:

```txt
app/globals.css
```

## Как выложить сайт онлайн
Самый простой способ:
1. Создать аккаунт на Vercel: https://vercel.com
2. Загрузить проект на GitHub.
3. В Vercel нажать `Add New Project`.
4. Выбрать этот репозиторий.
5. Нажать `Deploy`.

## Важно
Форма сейчас визуальная. Чтобы заявки реально приходили, нужно подключить отправку в Telegram, WhatsApp, email или CRM.
